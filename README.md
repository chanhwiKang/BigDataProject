# 프로젝트 진행사항 (C반 강찬휘)

안녕하세요, C반 강찬휘입니다.  
진행사항 말씀드리기 전, 프로젝트의 배경을 먼저 설명드립니다.

## 프로젝트 배경

1. 모든 데이터는 2015년부터 2020년까지 기준으로 합니다.
2. 세종시는 2019년도까지 데이터가 없어 제외하였습니다.
3. 외국인 비율과 범죄 수의 변화를 한눈에 보기 위해, 수업에서 배우지 않은 `twinx()` method를 사용하였습니다. (교수님이 사용 금지라고 하시면 제거 가능합니다.)
4. 한국인과 외국인의 데이터는 공공 데이터를 이용하였습니다.
5. 범죄 수 데이터는 경찰청에서 매년 배포하는 PDF 파일을 바탕으로 하드코딩하였습니다.
6. 가능한 상세하게 주석을 달았습니다.

## 2025년 11월 14일 기준 진행사항

0. `matplotlib` 및 기타 그래프에서 한글이 표시되지 않아, 한글 폰트를 지정하였습니다.
1. 한국인 데이터를 가공하였습니다.
2. 외국인 데이터를 가공하였습니다.
3. 범죄 수 PDF를 바탕으로 DataFrame을 생성하였습니다.
4. 한국인과 외국인 데이터를 이용하여 연도별/지역별 외국인 비율을 나타내는 DataFrame을 생성하였습니다.
5. 연도별/지역별 외국인 비율을 `lineplot`으로 시각화하였습니다.
6. `subplot`을 통해 대도시군, 중위권 도시군, 저인구 도시군으로 분류하여 연도별/지역별 외국인 비율을 `lineplot`으로 그렸습니다.
7. 연도별/지역별 범죄 수를 `lineplot`으로 시각화하였습니다.
8. `subplot`을 통해 대도시군, 중위권 도시군, 저인구 도시군으로 분류하여 연도별/지역별 범죄 수를 `lineplot`으로 그렸습니다.
9. `twinx()` method를 이용하여 범죄 수(막대 그래프)와 외국인 비율(선 그래프)을 함께 시각화하고, `subplot`을 통해 대도시군, 중위권 도시군, 저인구 도시군으로 분류하여 그렸습니다. (현재 조정 중)
